3.4 Perfiles

La opción de Perfiles esta disponible solamente para el Administrador del sistema. Cuando el administrador desee ver la lista de usuarios, crear o editar los perfiles de usuarios debe hacer click en esta opción. La primiera vista que se mostrará es el listado de usuarios del sistema con el botón de editar el cual abre el formulario para editar los usuarios.
 
 
 
 
 
Si el administrador desea crea un nuevo usuario debe presionar el botón nuevo usuario y se desplegará el formulario de Nuevo Usuario donde se completan todos los datos del usuario y se le asignan los permisos que se desean.
 
 
 
 
La ayuda en línea se crea con Dr.Explain
